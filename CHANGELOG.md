# История изменений

## 0.1.5 — 2026-07-20

- Мобильная и web-версии приведены к одному актуальному интерфейсу инструмента.
- Добавлена вкладка чтения диапазонов и сравнение преимущества диапазонов на флопе.
- После каждого решения диапазон открывается сразу с выделением сыгранной руки.
- Исправлены размеры карт, адаптивная компоновка тренировки и обозначения действий за столом.
- Песочница показывает все 54 импортированных спота; скрытые ответы на 3-бет и 4-бет доступны.
- Удалены уровни обучения, XP, рейтинги и остальные отвлекающие элементы.

## 0.1.0 — в разработке

- Expo SDK 57, React Native 0.86 и web static export.
- Provider-aware диапазоны, пресеты, статистика и очередь повторения.
- Persistence schema v2 с миграцией данных v1.
- Реальный Exam mode с настраиваемым лимитом ошибок.
- Публичные данные ограничены MIT-паками Pekarstas и Greenline.
- EV и bb/100 не отображаются как числа без необходимых исходных данных.
- Добавлены лицензии, manifest источников, EAS-профили, CI и сайт загрузки.
